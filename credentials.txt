=== CREDENCIAIS DO SISTEMA ADMIN ===

Usuário: admin
Senha: administrador

=== INSTRUÇÕES ===

1. Estas são as credenciais padrão do sistema
2. Para maior segurança, altere a senha após o primeiro login
3. O sistema usa localStorage para persistência
4. As senhas são hasheadas antes de serem armazenadas

=== SEGURANÇA ===

- O sistema bloqueia a conta após 5 tentativas de login
- A sessão expira após 1 hora de inatividade
- As senhas são hasheadas antes de serem armazenadas
- O sistema registra tentativas de login e bloqueios

=== SOLUÇÃO DE PROBLEMAS ===

Se o login não funcionar:
1. Acesse: /clear_cache.html
2. Clique em "Limpar Todo o Cache"
3. Tente fazer login novamente
4. Se ainda não funcionar, verifique o console do navegador (F12)
